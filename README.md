 PROJECT OVERVIEW
This project is a travel website developed as part of a programming assignment. The website provides users with information about different travel destinations, including available hotels, restaurants, and activities. It is designed to simulate a real-world travel booking platform.

 WEBSITE GOALS AND OBJECTIVES
To create a functional and user-friendly website
To demonstrate understanding of HTML and basic CSS
To allow users to explore travel destinations
To provide information on hotels, restaurants, and activities
To practice website structure and navigation


KEY FEATURES AND FUNCTIONALITY
Home page with introduction
About page with website information
Destinations page (Cape Town, Dubai, Kenya, Mauritius)
Sections for hotels, restaurants, and activities
Contact form for user interaction
Navigation menu for easy browsing


 TIMELINE AND MILESTONES
Week 1: Planning and structure of the website
Week 2: Creating HTML pages
Week 3: Adding content (destinations, images, text)
Week 4: Styling using CSS
Week 5: Testing and final improvements
